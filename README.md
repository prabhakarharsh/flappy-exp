# 🐦 Flappy Bird Clone

A simple and responsive **Flappy Bird** clone built using **HTML, CSS, and JavaScript**. The game recreates the classic gameplay where players control a bird, avoid obstacles, and try to achieve the highest score.

## 🎮 Demo



Example:
https://luminous-marshmallow-7af1c4.netlify.app/

---

## 🚀 Features

- Smooth bird movement and gravity physics
- Randomly generated pipes
- Collision detection
- Real-time score tracking
- Game Over screen
- Restart functionality
- Responsive design

---

- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/flappy-bird.git
```

2. Open the project folder

```bash
cd flappy-bird
```

3. Open `index.html` in your browser.

No installation or dependencies required.
